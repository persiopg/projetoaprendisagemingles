# Atribuição (Wiktionary / Kaikki)

Este projeto pode gerar dados em [web/src/data/mostCommonEnglishWords2000.generatedOverrides.ts](src/data/mostCommonEnglishWords2000.generatedOverrides.ts) a partir de um dump offline do Wiktionary disponibilizado pela Kaikki.org.

- Fonte: Wiktionary (conteúdo colaborativo) via Kaikki.org (formato JSONL)
- Licença do conteúdo do Wiktionary: **CC BY-SA**

Se você incluir no repositório conteúdo derivado do Wiktionary (por exemplo: traduções, exemplos, definições/glosses), você precisa cumprir as exigências de atribuição e *share-alike* da CC BY-SA.

Recomendação prática:
- Mantenha o dump bruto fora do git (veja [web/.gitignore](.gitignore)).
- Registre a origem do dump e a data em que foi obtido.
- Se publicar/redistribuir o material derivado, revise se a licença do seu repositório está compatível com CC BY-SA.
